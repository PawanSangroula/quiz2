# Campus Companion (Quiz #2)

A small Java Swing GUI app to practice GitHub collaboration.

## How to Run

### Option A: Terminal (works everywhere)
From the project root:

**Compile**
```bash
javac -d out src/app/MainApp.java src/app/features/*.java
```
**Run**
```bash
java -cp out app.MainApp
```

### Quiz #2 Instructions (Per Student)
#### 1. Clone the repo
```bash
git clone <REPO_URL>
cd CampusCompanion
```
